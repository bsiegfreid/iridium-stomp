# Rust Codespace

This repository is configured for development in a GitHub Codespace or VS Code Dev Container with Rust support.

## Features
- Rust toolchain (via official dev container image)
- `rust-analyzer` and `CodeLLDB` VS Code extensions
- `clippy` and `rustfmt` installed for linting and formatting

## Getting Started
1. Open this repository in GitHub Codespaces or VS Code with the Dev Containers extension.
2. The environment will automatically install Rust and recommended extensions.
3. Start coding in Rust!

## Customization
You can modify `devcontainer.json` to add more tools or extensions as needed.
